1. First Clone this repo
2. Second open terminal and navigate to Backend folder ("cd Backend")
3. Install packages ("npm install")
4. Connect Database
5. Open Xampp Control Panel. Start MYSQL and Apache2 Service
6. Create a database with name postcodedb
7. import the database code from "Backend/DataBase/postcodedb.sql"
8. Open Editor and type command "npm start" to run backend
9. Now Open Frontend folder and open index.js to view project
